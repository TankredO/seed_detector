# seed_detector

## Installation

```bash
pip install git+https://github.com/TankredO/seed_detector
```

## Usage

```bash
seed_detector -h
```

### Extraction

```bash
seed_detector single -h
```

or

```bash
seed_detector multi -h
```

### Alignment

```bash
seed_detector align -h
```

### Measurement

```bash
seed_detector measure -h
```
